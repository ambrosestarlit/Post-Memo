# 📝 Memonto

SNS風に思いついたことをメモできるブラウザアプリです。

## ✨ 特徴

- 📱 **スマホ専用** - PCでも使えますがスマホ向けにデザインされています。
- 💾 **オフライン保存** - LocalStorageでデータを保存（キャッシュクリアするまで永続）
- 📷 **画像添付** - 複数の画像を同時に投稿可能
- 🏷️ **ハッシュタグ** - #タグでカテゴリ分け・検索が可能
- 🎨 **カスタマイズ** - 5種類のカラーテーマ、アイコン・背景画像の変更
- 📥 **バックアップ** - JSONファイルでデータのエクスポート/インポート
- 📱 **PWA対応** - ホーム画面に追加してアプリのように使用可能

## 🚀 使い方

### GitHub Pagesでの公開方法

1. GitHubで新しいリポジトリを作成
2. このフォルダ内のすべてのファイルをリポジトリにアップロード
3. Settings > Pages > Source を "main" ブランチに設定
4. 公開されたURLにアクセス

### ローカルでの使用

1. `index.html` をブラウザで開く
   - または、ローカルサーバーを起動:
   ```bash
   python -m http.server 8000
   ```
   - ブラウザで `http://localhost:8000` を開く

### スマホのホーム画面に追加

#### iPhone (Safari)
1. Safariでアプリを開く
2. 画面下部の共有ボタンをタップ
3. 「ホーム画面に追加」を選択

#### Android (Chrome)
1. Chromeでアプリを開く
2. 右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択

## 🎯 機能一覧

### 投稿機能
- テキスト投稿
- 複数画像の同時投稿
- URL自動リンク化
- ハッシュタグ自動認識

### 閲覧機能
- タイムライン表示
- 画像のダウンロード（個別・一括）
- テキストのコピー
- 投稿の削除

### 検索機能
- ハッシュタグによる絞り込み
- ハッシュタグ一覧表示
- 投稿数のカウント表示

### カスタマイズ
- **5種類のテーマ**
  - チョコレート（デフォルト）
  - 桜
  - オーシャン
  - フォレスト
  - サンセット
- ユーザーアイコンの変更
- 背景画像の設定

### データ管理
- JSONファイルでエクスポート
- JSONファイルからインポート（上書き/追加選択可）
- 全データの削除

## 💡 使用例

### ハッシュタグの使い方
```
今日のタスク完了！ #仕事 #達成感
```

### URL付き投稿
```
面白い記事を見つけた
https://example.com/article
#参考資料
```

### 画像付きメモ
```
今日のランチ 🍜
#グルメ #ラーメン
```
（画像を添付）

## 🔒 プライバシー

- すべてのデータはブラウザのLocalStorageに保存されます
- サーバーへのデータ送信は一切行われません
- 完全にオフラインで動作します

## 🎨 カラーテーマ

2. **桜** - 優しいピンクベース
